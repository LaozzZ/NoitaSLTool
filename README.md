# NoitaSLTool
基于C++实现的Noita存档读档工具
使用C++23编译  
仅Windows系统可用
## 配置指南（必读）
* 游戏设置
  
  * 在Noita主界面中点击 **选项** - **常规**，在**在线**中找到**steam云存档**选项，将其设置为**关**。
  * 游戏中的**保存与退出**选项会直接退出游戏，请务必在游戏退出的状态下进行存档/读档操作。
* 路径配置
  * 存档的保存路径需自己创建，在想要保存的位置新建一个文件夹作为保存路径即可，如`E:\save`，注意使用英文路径。  
  * 程序在每次启动时会检测配置的保存路径和Noita路径是否合法，第一次打开程序时，由于没有配置路径，会进入路径配置环节，其中**Noita存档路径**为游戏中存档所在的路径，具体路径为`C:\Users\你的用户名\AppData\LocalLow\Nolla_Games_Noita\save00`。**存档保存路径**为你在上一步新建的文件夹，例如我在E盘内创建了一个名为**save**的文件夹，那么**存档保存路径**即为`E:\save`。
    
    **注意：** 路径结尾请勿出现 **'/'** 或 **'\\'**。

  * 程序检测路径是否合法的逻辑为**路径是否为文件夹**，因此即使你输入的Noita存档路径错误，只要它是文件夹，程序就会判断通过，因此可能会出现路径出错从而无法存档读档的问题，请务必仔细检查路径是否正确，如果需要修改路径，可以输入指令`changesourcepath` 或 `sourcepath`来修改**Noita存档路径**，输入指令`changesavepath` 或 `savepath`来修改**存档保存路径**，具体请见下方操作说明。
## 操作说明
**说明：** 程序中的所有指令输入部分均不分大小写，可随意输入。  
**注意：** 请在游戏退出的状态下进行存档/读档操作。  

* **退出程序：** 指令`quit` 或 `exit`
  * 当然你也可以直接关闭窗口。
* **帮助：** 指令`help`
  * 打印所有的指令，简述各个指令的作用。
* **存档：** 指令`save`
  * 输入指令后可输入**1 ~ n**的整数作为存档的编号，接着输入存档的名称即可完成存档，若编号已被占用则会弹出是否覆盖存档的选项。
* **快速存档：** 指令`qsave`
  * 程序会自动寻找最小且为空的编号作为存档的编号，并将存档命名为QuickSave + 编号。
* **自动存档（重要）：**
  * 可以使用指令`changeautosave` 或 `autosave` 来切换自动存档功能的开启或关闭，在指令`info` 和 `config` 中也会显示自动存档功能是否开启，自动存档开启后，每当你退出游戏时，程序会将当前的游戏存档保存至0编号处，存档名称为**AutoSave**，并且会覆盖掉先前的自动存档。  
  * 因此在游戏时，你可以随时退出游戏再重进，程序会保存你上次退出游戏时的存档，当你死后想回档时，**请勿直接退出游戏**，**请勿直接退出游戏**，**请勿直接退出游戏**，而是在程序中使用指令`changeautosave` 或 `autosave`将自动存档功能关闭，接着退出游戏，使用读档操作读入上次的自动保存存档，再进入游戏就会回到上次的自动保存位置了。
* **读档：** 指令`load`
  * 输入指令后可输入**0 ~ n**的整数选择存档编号进行读档，其中0为自动保存存档。
* **快速读档：** 指令`qload`
  * 程序会自动读档最近一次保存的存档，可用于快速读取自动保存存档。
* **删除存档：** 指令`del`
  * 输入指令后可输入**0 ~ n**的整数选择存档编号进行删除存档，其中0为自动保存存档。
* **删除全部存档：** 指令`delallsaves`
  * 删除存档目录内的所有存档。
* **存档信息：** 指令`saveinfo` 或 `info`
  * 打印保存的所有存档信息，同时显示排序方式和自动保存状态。
* **配置信息：** 指令`config`
  * 打印Noita存档路径、存档保存路径、排序方式和自动保存状态。
* **更改排序方式：** 指令`changecmpfunc` 或 `cmp`
  * 修改存档信息打印时的排序方式（按编号从小到大 / 按时间从近到远）。
* **更改存档保存路径：** 指令`changesavepath` 或 `savepath`
  * 此操作会将原路径中的存档文件转移到更改后的路径中，并清空原路径的文件夹。
* **更改Noita存档路径：** 指令`changesourcepath` 或 `sourcepath`
* **更改自动存档状态：** 指令`changeautosave` 或 `autosave`
  * 可以随时更改，即使是在游戏中。
## 安装与运行
* 确保你的操作系统是Windows。
* 下载[Release](https://github.com/LaozzZ/NoitaSLTool/releases/tag/1.0)中的压缩包**static-NoitaSLTool.rar**并解压。Release中的程序使用静态链接编译，即使没有配置C++环境变量也可以使用。
* 如果想在本地进行调试运行，也可以克隆或下载此完整仓库到本地。
